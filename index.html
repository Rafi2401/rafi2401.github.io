YouTube Songs Analysis
Using Power BI

The internship project focuses on analyzing YouTube songs data using Power BI to create insightful visualizations and reports. The dataset includes key attributes such as video ID, channel title, song title, description, tags, publication date, view count, like count, favorite count, comment count, video duration, definition, and caption details. The project objectives involve cleaning and preparing the data, conducting exploratory data analysis (EDA) to uncover patterns in user engagement and popularity, analyzing content across channels, exploring temporal trends, and deriving user engagement insights. The deliverables include interactive Power BI dashboards and reports that provide recommendations for content creators and stakeholders to optimize their YouTube song videos based on identified trends and preferences.

Hotel Reservation Analysis
Using MySQL

The Hotel Reservation Analysis project involves using SQL to analyze a hotel reservation dataset, enhancing data analysis skills in a practical context. The project aims to derive insights into guest preferences, booking trends, and operational factors affecting the hotel industry. The dataset includes attributes such as Booking_ID, number of adults and children, weekend and weekday nights, meal plans, room types, lead time, arrival date, market segment, average price per room, and booking status. Participants will write SQL queries to answer specific questions, including the total number of reservations, the most popular meal plan, average prices, common room types, lead times, booking statuses, and market segments. The analysis will ultimately assist in making informed decisions to improve guest experiences and optimize hotel operations.

Android Malware Detection
Using Parallel Machine Learning Classifiers

This project aims to evaluate the performance of the Sequential Minimal Optimization (SMO) algorithm and its parallel variants, PSMO-2 and PSMO-4, in detecting malware on Android devices using the DREBIN dataset. In this study, the SMO algorithm is employed to train a Support Vector Machine (SVM) model, which is known for its effectiveness in handling large-scale classification tasks. By utilizing parallelization techniques, specifically PSMO-2 and PSMO-4, the project seeks to enhance the speed and efficiency of the model training process without compromising accuracy. The analysis results demonstrate how these optimization techniques can significantly accelerate malware detection while maintaining strong classification performance in identifying malicious applications.

Jun 13, 2021

IDENTIFYING INFLUENCERS ON TWITTER FOR COVID-19 EDUCATION AND VACCINATION
USING SOCIAL NETWORK ANALYSIS

This project focuses on Social Network Analysis (SNA) using Twitter user data to identify influencers and understand interaction patterns within a specific network. By applying various centrality metrics such as degree centrality, betweenness centrality, and closeness centrality, the analysis aims to uncover key users who play influential roles in spreading information. The project also explores the community structure of the network, grouping users based on their interaction density. Visualizations of the social network are generated to provide a clearer understanding of the connectivity and influence patterns. The insights gained from this analysis can be useful for various applications, such as targeted marketing, opinion leader identification, and understanding the dynamics of information dissemination in social media.
